Генератор умеет "цеплять" конфигурацию joomla и правильно обрабатывать сессии для жумлы.
Конфигурация формы находится в файле /config/form-casting.php (пример).
Скрипт обработки находится в файле /casting.php (пример).
Скрипт поддерживает мультиязычность, язык должен быть определён в константе _LANG в основном файле (casting.php).
